Adds a custom dashboard widget to display scheduled posts and custom post types.

== Description ==

A simple wordpress widget that adds a custom widget to Wordpress dashboard. 
It displays scheduled items, being them posts or any custom post type.
Each entry presents date of publication, title with link to the edit page, category, post type and a button to open a preview.

You can filter only the post types you want to see. The widget can remember your choice the next time you open your dashboard.

Author: Rici86 (rici86.com)
Github repository: https://github.com/rici86/scheduled_dashboard_widget

== Installation ==

1. Upload the `scheduled-posts-dashboard-widget` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin
